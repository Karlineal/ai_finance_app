
## 七、注意事项

1. **API 密钥不涉及**：本功能无需 DeepSeek/PaddleOCR
2. **序列化已支持**：`Converters.kt` 已覆盖 UUID、BigDecimal、LocalDate、Instant
3. **Hilt**：ViewModel 用 `@HiltViewModel`，Repo 实现用 `@Inject constructor` + `@Singleton`
4. **主题**：不硬编码颜色，使用 `MaterialTheme.colorScheme`
5. **日期**：统一 `java.time.LocalDate` / `Instant`，不用 `java.util.Date`
6. **金额**：使用 `BigDecimal`，不用 `Double`/`Float`
7. **提交规范**：Conventional Commits，如 `feat(savings-goal): add domain model and entity`
