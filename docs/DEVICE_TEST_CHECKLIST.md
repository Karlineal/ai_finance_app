# iCookie 真机测试清单

## 测试前准备

### 1. 构建 Release APK
```bash
flutter build apk --release
```

### 2. 安装到测试设备
```bash
# 方式1: 使用 adb
adb install build/app/outputs/flutter-apk/app-release.apk

# 方式2: 直接传输 APK 到手机安装
```

### 3. 准备测试数据
- 至少准备 5-10 笔交易记录（不同日期、分类）
- 设置月度预算
- 创建一个存钱计划
- 添加一个订阅

---
